Learning Machine Learning algorithms. First a Linear Regression model, then a multilayer network for determining Iris species

MLP is a multi-layer perceptron to recognize a number given the inputs of a 7-segment digitial clock.

DLM.ipynb is a step by step development of learning how to use PyTorch: Tensor creation, extracting data from tensors, creating basic neurons and layers, understanding the grad() function, loss() function, optimizer(). Then I develop a multi-layer perceptron or feed-forward network on the iris-data set to classify species. It is a supervised machine learning algorithm.
